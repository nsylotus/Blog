# Order By子句

ORDER BY 关键字用于对结果集按照一个列或者多个列进行排序。默认是升序(ASC)，如果需要降序则使用(DESC)关键字.

order by子句在对某一列进行排序时可以使用这一列所对应的数字，也可以使用这一列所对应的列名。后面跟asc或者desc即可。
